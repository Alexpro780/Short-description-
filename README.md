Short description (EN): A basic on-chain price oracle mock with an owner-settable price and a simple read interface.
- app.py  
  Основной исполняемый файл на Python. Содержит:
  - Логику генерации ссылок на блокчейн-эксплореры.
  - Простейший CLI на argparse.
  - Интерактивный режим (вопросы в консоли).
  - Логирование запросов в файл multi_chain_logs.json.
  - Вывод сводки по логам.
